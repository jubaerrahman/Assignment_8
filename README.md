## 🌐 Live Site

## 📌 Description

SkillSphere is a simple online learning platform where users can browse courses, view details, and manage their profile.

## ✨ Features

* Login & Register 
* Protected Course Details Page
* Course Search Functionality
* Profile Page
* Hero Slider (Swiper)
* Loader & 404 Page
* Responsive Design

## 🛠️ Technologies

* Next.js
* Tailwind CSS
* Swiper.js
* React Icons
* React Hot Toast

## 🚀 Run Locally

```bash
npm install
npm run dev
```
